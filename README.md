# RustSnake

## Simple snake game i made to learn piston basics

### Requirements

- Rust Installation

### Build from source

1. Clone the repo

   ```sh
   git clone https://github.com/DavinderJolly/RustSnake.git
   ```

2. Change directory into the cloned repo

   ```sh
   cd RustSnake
   ```

3. Build using cargo

   ```sh
   cargo build --release
   ```
